# Canonical Promotion Checklist

Use [slice-257-canonical-promotion-checklist.md](/home/pboling/src/kettle-rb/spec/slices/slice-257-canonical-promotion-checklist.md)
when preparing a future canonical promotion for a family-scoped suite.

The expected checklist is:

- identify the suite name and target canonical set
- identify the canonical review-request identity surface
- identify reviewed-default context behavior
- identify replay-compatibility behavior
- identify manifest or fixture membership changes
